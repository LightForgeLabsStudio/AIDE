# Implementation Agent - Step 7: Report Back

## Purpose
Communicate completion clearly vs the spec, including testing and any deviations.

## Prerequisites
- Step 6 complete (PR is ready for review).

## Step Actions

After PR is ready, report to user:

```markdown
**Implementation complete**

**PR:** #<number> (ready for review)
**Issue:** #<issue> (will close on merge)

**Summary:**
- [What was implemented]
- [What was tested]

**Deviations from spec:** [None / list any]
```

## Exit Criteria
- Status is reported with PR link/number, testing, and deviations (if any).
